# CODE OF CONDUCT

We want this repository to be a respectful, practical, and welcoming place to collaborate.

## Our Standards

Examples of behavior that help the community:
- showing empathy and patience
- giving constructive technical feedback
- respecting different skill levels and backgrounds
- focusing discussions on facts, evidence, and improvements
- reporting security or privacy concerns responsibly

Examples of behavior that are not acceptable:
- harassment, threats, or personal attacks
- publishing private information without permission
- discriminatory or hateful language
- spam, malicious payloads, or intentionally harmful instructions
- repeated bad-faith disruption of issues, pull requests, or discussions

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this code of conduct. Repeated or severe violations may result in temporary or permanent bans from project spaces.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainer through GitHub or another private contact channel.
