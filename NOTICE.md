# NOTICE

`xiaohongshuritter` is an independently maintained repository for enterprise-oriented Xiaohongshu MCP deployment.

## Upstream Reference

This project references the open-source repository [xpzouying/xiaohongshu-mcp](https://github.com/xpzouying/xiaohongshu-mcp) for design and implementation ideas.

On top of that baseline, this repository adds and organizes:
- enterprise multi-tenant and multi-account routing
- concurrency governance for shared accounts
- OpenClaw-oriented deployment templates
- Docker and source deployment documentation
- ARM64 and bundled-browser delivery workflows

## Dependency Notice

This repository also depends on the related browser helper package `github.com/xpzouying/headless_browser` through Go modules.
